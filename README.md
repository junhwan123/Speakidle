# Speakidle (Speaki + Wordle)

스피키로 워들을 플레이해봅시다<br>
Play Wordle with Speaki

##  게임 소개
Speakidle은 Wordle의 규칙을 그대로 따릅니다.<br>
다만 정해진 단어만을 사용하는 Wordle과 달리, Speakidle은 랜덤한 조합을 사용합니다.<br>
난이도 조절을 위해 답은 5개의 중복 없는 스피키들로 구성됩니다.<br>
(설정을 통해 조절 가능합니다.)

Speakidle follows the rules of Wordle.<br>
Unlike the original Wordle which uses a fixed dictionary, Speakidle uses random combinations.<br>
For balanced difficulty, the answer consists of 5 unique Speaki symbols.

## 플레이 방법
*   **키보드 입력**: 페이지 아래에 떠 있는 가상 키보드를 통해 입력합니다.<br>
가상 키보드가 아닌 기록에 있는 아이콘을 눌러도 입력이 가능합니다.
*   **정답 추측하기**: 입력한 아이콘의 테두리 색을 통해 정답을 추측할 수 있습니다.
    *   **초록색**: 아이콘과 위치가 모두 일치합니다.
    *   **노란색**: 아이콘은 정답에 포함되지만, 위치가 다릅니다.
    *   **회색**: 해당 아이콘은 정답에 포함되지 않습니다.
*   **정답 맞추기**: 정답을 맞추면 정답 창을 통해 입력한 기록과 정답을 확인할 수 있습니다.<br>
저장을 통해 해당 기록을 이미지 파일로 저장할 수 있습니다.

## 게임 설정
*   **정답 재설정**: 지금 정답을 확인하고 새로운 게임을 시작합니다.
*   **정답 직접 설정**: 원하는 정답을 직접 입력하여 게임을 시작합니다. 입력한 정답은 표시되지 않습니다.
*   **설정**
    *   **정답 개수**: 정답 아이콘의 개수를 수정합니다.
    *   **중복 글자 허용**: 정답 아이콘의 중복을 허용합니다.

## How to Play
*   **Keyboard Input**: Input using the virtual keyboard floating at the bottom of the page.<br>
You can also input by clicking on the icons in the log instead of the virtual keyboard.
*   **Guessing the Answer**: You can guess the answer based on the border color of the icons.
    *   **Green**: The icon and position both match.
    *   **Yellow**: The icon is included in the answer, but the position is different.
    *   **Gray**: The icon is not included in the answer.
*   **Winning the Game**: When you guess the answer, you can check your log and the answer in the result window.<br>
You can save the record as an image file through the save button.

## Game Settings
*   **Reset Answer**: Check the current answer and start a new game.
*   **Custom Answer**: Start the game by manually entering the desired answer. The entered answer is not displayed.
*   **Settings**
    *   **Answer Length**: Modify the number of icons in the answer.
    *   **Double letters**: Allow double icons in the answer.
